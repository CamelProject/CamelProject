CamelError

CamelError is a division of CamelProject that develops error pages. Below is a table of HTTPS table codes.

Code Description
1XX Information codes (not errors).
2XX Successful request codes (not errors).
3XX Redirection codes (some are errors).
4XX An error on the client side.
5XX An error on the server side.
